Semester Project 2023

This is an application for tracking staff and delivery drivers for the ficticious company WeDeliverTech™. The application is a receptionist board with tables for staff members and delivery drivers. The staff table is populated with random staff members using randomuser.me. Delivery drivers are manually entered and go through a verification process to check that the entered information is valid. The application checks if employees are late and alerts the user if they are. The application is written using vanilla javascript and jquery v.3. Styling is done using bootstrap v.5 and custom css.

Grade: A.

Built with

Javascript, Bootstrap, Jquery, HTML, CSS
